API Status

The API server is running. To check the health of the API, visit /.netlify/functions/api/health

Available Endpoints

Authentication

POST /.netlify/functions/api/auth/login - User login

POST /.netlify/functions/api/auth/register - User registration

Songs

GET /.netlify/functions/api/songs - Get all songs

Testing

GET /.netlify/functions/api/test - Test endpoint

GET /.netlify/functions/api/health - Health check
